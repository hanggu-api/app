App Name & Vision

101 Service: The Hybrid Marketplace

A unified ecosystem merging on-demand mobility with localized professional services. 101 Service acts as the bridge between transportation and home/fixed-site services, providing a seamless user experience for both moving people and bringing expertise to the user's doorstep.

Key User Flows

1. Mobility Uber Style

  • Instant vehicle request based on current GPS.
  • Real-time driver tracking on map.
  • Fare estimation and in-app payment.

2. Fixed Services Location Based

  • Discover nearby salons, barbers, or clinics.
  • In-app calendar for direct booking.
  • User reviews and portfolio viewing.

3. Mobile Services (In-Home) Priority Flow

  1. Booking: Request service + Pay 30% upfront.
  2. Tracking: Provider en-route visualization.
  3. Arrival: Push notification + User provides safety code.
  4. Completion: Finalize service + Automated 70% payment.

Technical Stack Suggestions

Geolocation

Google Maps API / Mapbox for real-time provider tracking and spatial indexing.

Notifications

Firebase Cloud Messaging (FCM) for instant arrival and payment alerts.

Payments

Stripe Connect for automated split-payments (30/70) and escrow handling.

Authentication

OTP via SMS / Bio-metrics (FaceID) for secure provider verification.
